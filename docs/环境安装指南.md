# Node.js 和 Maven 安装指南

## 一、Node.js 安装

### 方法一：官网下载安装（推荐）

1. **访问Node.js官网**
   - 打开浏览器，访问：https://nodejs.org/zh-cn/
   - 或者直接访问：https://nodejs.org/

2. **下载安装包**
   - 官网会自动检测您的系统，推荐下载 **LTS版本**（长期支持版本，更稳定）
   - 点击绿色的 "推荐给大多数用户" 按钮下载
   - 文件名类似：`node-v18.17.0-x64.msi`（Windows 64位）

3. **安装Node.js**
   - 双击下载的 `.msi` 安装包
   - 点击 "Next" 继续
   - 接受许可协议，点击 "Next"
   - 选择安装路径（默认即可：`C:\Program Files\nodejs\`），点击 "Next"
   - **重要**：勾选 "Automatically install the necessary tools" 选项
   - 点击 "Install" 开始安装
   - 等待安装完成，点击 "Finish"

4. **验证安装**
   - 按 `Win + R`，输入 `cmd`，打开命令提示符
   - 输入以下命令检查版本：
     ```bash
     node -v
     npm -v
     ```
   - 如果显示版本号（如 `v18.17.0` 和 `9.6.7`），说明安装成功

### 方法二：使用包管理器安装（可选）

**使用 Chocolatey（需要管理员权限）**：
```bash
# 以管理员身份运行PowerShell，然后执行：
choco install nodejs
```

### 配置npm镜像（可选，但强烈推荐）

由于国内访问npm官方源较慢，建议配置淘宝镜像：

```bash
# 设置npm镜像源
npm config set registry https://registry.npmmirror.com

# 验证配置
npm config get registry
```

## 二、Maven 安装

### 方法一：官网下载安装（推荐）

1. **访问Maven官网**
   - 打开浏览器，访问：https://maven.apache.org/download.cgi
   - 或者：https://maven.apache.org/

2. **下载Maven**
   - 找到 "Files" 部分
   - 下载 **Binary zip archive**（不要下载Source）
   - 文件名类似：`apache-maven-3.9.4-bin.zip`
   - **注意**：确保下载的是最新稳定版本（3.6+）

3. **解压Maven**
   - 将下载的zip文件解压到合适的位置
   - 建议解压到：`C:\Program Files\Apache\maven\`
   - 或者：`D:\Tools\maven\`
   - 解压后文件夹结构应该是：
     ```
     C:\Program Files\Apache\maven\
     └── apache-maven-3.9.4
         ├── bin
         ├── boot
         ├── conf
         └── lib
     ```

4. **配置环境变量**

   **Windows 10/11**：
   
   a. **配置MAVEN_HOME**
   - 右键 "此电脑" → "属性"
   - 点击 "高级系统设置"
   - 点击 "环境变量"
   - 在 "系统变量" 区域，点击 "新建"
   - 变量名：`MAVEN_HOME`
   - 变量值：Maven的解压路径（如：`C:\Program Files\Apache\maven\apache-maven-3.9.4`）
   - 点击 "确定"
   
   b. **配置PATH**
   - 在 "系统变量" 区域，找到 `Path` 变量，点击 "编辑"
   - 点击 "新建"，添加：`%MAVEN_HOME%\bin`
   - 或者直接添加完整路径：`C:\Program Files\Apache\maven\apache-maven-3.9.4\bin`
   - 点击 "确定" 保存所有更改

5. **验证安装**
   - **重要**：关闭所有已打开的命令提示符窗口（环境变量需要重新打开才能生效）
   - 重新打开命令提示符（`Win + R` → 输入 `cmd`）
   - 输入以下命令：
     ```bash
     mvn -v
     ```
   - 如果显示Maven版本信息（如 `Apache Maven 3.9.4`），说明安装成功

### 方法二：使用包管理器安装（可选）

**使用 Chocolatey（需要管理员权限）**：
```bash
# 以管理员身份运行PowerShell
choco install maven
```

**使用 Scoop（需要先安装Scoop）**：
```bash
scoop install maven
```

### 配置Maven镜像（可选，但强烈推荐）

Maven默认从国外下载依赖较慢，建议配置阿里云镜像：

1. **找到Maven配置文件**
   - 位置：`C:\Program Files\Apache\maven\apache-maven-3.9.4\conf\settings.xml`
   - 或者在用户目录：`C:\Users\你的用户名\.m2\settings.xml`（如果没有就创建一个）

2. **编辑settings.xml**
   - 用文本编辑器（记事本、Notepad++等）打开 `settings.xml`
   - 找到 `<mirrors>` 标签（如果没有就在 `<settings>` 标签内添加）
   - 添加以下内容：

   ```xml
   <settings>
     <mirrors>
       <mirror>
         <id>aliyunmaven</id>
         <mirrorOf>*</mirrorOf>
         <name>阿里云公共仓库</name>
         <url>https://maven.aliyun.com/repository/public</url>
       </mirror>
     </mirrors>
   </settings>
   ```

3. **保存文件**

## 三、JDK 安装（如果没有）

Maven需要JDK才能运行，如果没有安装JDK：

1. **下载JDK**
   - 访问Oracle官网：https://www.oracle.com/java/technologies/downloads/
   - 或者使用OpenJDK：https://adoptium.net/
   - 下载 **JDK 8 或 JDK 11**（LTS版本）
   - 选择Windows x64版本

2. **安装JDK**
   - 运行安装程序
   - 按照提示安装（建议使用默认路径）

3. **配置JAVA_HOME环境变量**
   - 右键 "此电脑" → "属性" → "高级系统设置" → "环境变量"
   - 新建系统变量：
     - 变量名：`JAVA_HOME`
     - 变量值：JDK安装路径（如：`C:\Program Files\Java\jdk1.8.0_XXX`）
   - 编辑 `Path` 变量，添加：`%JAVA_HOME%\bin`

4. **验证安装**
   ```bash
   java -version
   javac -version
   ```

## 四、完整安装验证

安装完成后，打开新的命令提示符窗口，依次执行以下命令验证：

```bash
# 验证Java
java -version

# 验证Maven
mvn -v

# 验证Node.js
node -v

# 验证npm
npm -v
```

所有命令都应该显示版本号，没有错误。

## 五、常见问题

### Node.js相关问题

**Q1: npm install 速度很慢怎么办？**
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com
```

**Q2: 提示 'node' 不是内部或外部命令？**
- 检查环境变量PATH是否包含Node.js的安装路径
- 重启命令提示符窗口

### Maven相关问题

**Q1: 提示 'mvn' 不是内部或外部命令？**
- 检查MAVEN_HOME环境变量是否正确配置
- 检查PATH环境变量是否包含 `%MAVEN_HOME%\bin`
- **重要**：配置环境变量后必须重新打开命令提示符

**Q2: Maven下载依赖很慢？**
- 配置阿里云镜像（见上面配置步骤）
- 或者使用VPN

**Q3: 找不到settings.xml文件？**
- 可以自己创建：在 `C:\Users\你的用户名\.m2\` 目录下创建 `settings.xml`
- 或者在Maven安装目录的 `conf` 文件夹下复制 `settings.xml` 到用户目录

### 环境变量配置技巧

1. **查看环境变量**：
   - 命令提示符中输入 `echo %PATH%` 查看PATH变量
   - 输入 `echo %JAVA_HOME%` 查看JAVA_HOME变量

2. **环境变量不生效**：
   - 配置环境变量后，必须关闭并重新打开命令提示符
   - 或者重启电脑

## 七、快速安装脚本（PowerShell）

如果您熟悉PowerShell，可以使用以下脚本快速安装（需要管理员权限）：

```powershell
# 安装Chocolatey（如果没有）
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# 使用Chocolatey安装Node.js和Maven
choco install nodejs maven -y
```

**注意**：使用脚本安装需要管理员权限，且需要先安装Chocolatey。
